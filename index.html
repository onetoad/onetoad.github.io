用Hugo搭建个人博客

在本地搭建博客 Windows下载安装Hugo 在hugo_release页面下载安装hugo_extended_xxx_Windows-64bit.zip 解压，把hugo.exe移动到C:\Software\hugo\hugo.exe 将C:\Software\hugo\加到PATH 重启终端，运行hugo version查看版本 依据文档光速搭建博客 打开Hugo官网的QuickStart，从Step 2开始根据提示进行建站。 创建一个新站点 quickstart为站点名字，自己取一个吧 hugo new site quickstart 添加一个皮肤 cd quickstart git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke echo 'theme = "ananke"' >> config.toml 写第一篇文章 hugo new posts/my-first-post.md md文件在相对路径quickstart\content\posts\下，可以打开编辑 其中draft为true则是草稿，false则是正文 本地启动Hugo服务器 hugo server -D 在浏览器中打开http://localhost:1313/ 至此，站点已经建完 但是我们会在后文将站点部署在 GitHub Pages上，因此还需要修改网站的配置文件并生成静态页面。 部署到GitHub Pages上 在GitHub上创建一个Repo 这个repo命名为xxx.github.io(xxx为你的Github全小写用户名) 修改配置文件 打开config.toml,修改baseURL baseURL = "https://xxx.github.io/" 建立静态页面 在你的站点根目录执行 hugo -D 默认情况下，输出将位于目录.
